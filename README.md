# IT-DEVELOPER
To-Do List Application using Python and Flask

## Project Overview
This is a simple To-Do List application built using Python and Flask.  
It allows users to register, log in, and manage their tasks securely.

## Features
- User authentication (login & registration)
- Add, delete, and complete tasks
- Task status tracking (pending/completed)
- Data stored using SQLite database
- Responsive and simple UI

## Technologies Used
- Python
- Flask
- SQLite
- HTML

## Future Enhancements
- Drag and drop task reordering using JavaScript
- Improved UI with CSS
- Deployment on cloud platform

## How to Run
1. Install Flask
2. Run `python app.py`
3. Open browser at `http://localhost:5000`
